# 유나이트 2019 트레이닝 데이 자료실
## "쉽고 빠르게 개발하는 모바일 슈팅게임"

![](https://github.com/IndieGameMaker/Unite2019/blob/master/image_readtop.png)

### 유니티 설치 버전 및 OS

- 2019.1.x 버전 이상
- 윈도우/맥OS
- Visual Studio 또는 Visual Studio Code(권장)

### 빌드 디바이스

- 안드로이드 4.4 버전 이상 권장
- 아이폰 6 이상 권장 (아이폰 빌드시 XCode 컴파일러가 설치된 맥북이 반드시 필요합니다.)
